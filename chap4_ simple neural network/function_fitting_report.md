# 函数拟合报告（ReLU 网络）

## 1. 函数定义

目标函数定义为：

\[
f(x)=\sin(2x)+0.3\cos(5x)+0.1x^2
\]

该函数同时包含低频、高频和二次项，能较好验证网络的非线性拟合能力。

## 2. 数据采集

- 训练集：在区间 `[-3, 3]` 上均匀随机采样 `512` 个点；
- 测试集：在区间 `[-3, 3]` 上等间隔采样 `256` 个点；
- 标签均由目标函数直接计算得到。

## 3. 模型描述

采用三层全连接 ReLU 网络（PyTorch 实现）：

- `Linear(1,64) + ReLU`
- `Linear(64,64) + ReLU`
- `Linear(64,1)`

优化器为 Adam，学习率 `1e-3`，损失函数为均方误差（MSE），训练轮数 `2000`。

## 4. 拟合效果

- 训练过程中训练集与测试集 MSE 均稳定下降；
- 训练结束后测试集 MSE 较低，预测曲线与目标函数曲线基本重合；
- 输出图像保存为 `fitting_result.png`（由脚本自动生成）。

## 5. 理论依据

多层前馈网络的通用逼近理论指出：在紧致集（有界闭集）上，只要激活函数满足一定非线性条件，单隐层网络即可在任意精度下逼近连续函数。ReLU 虽然是分段线性函数，但已有文献证明由 ReLU 构成的两层（输入层-隐层-输出层）网络同样具备通用逼近能力。其直观原因是：ReLU 网络可以通过大量分段线性基函数的加权组合，对目标函数进行局部拼接逼近；当隐层宽度增加时，逼近误差可进一步减小。

本实验在区间 `[-3, 3]` 上对目标函数进行采样并训练 ReLU 网络，测试集上预测曲线与真实曲线基本重合，实验现象与上述理论结论一致，即两层 ReLU 网络能够有效逼近该有界区间上的连续函数。
